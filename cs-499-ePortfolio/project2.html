Enhancement 2 - Algorithms and Data Structures

About This Artifact

For my second enhancement, I chose to enhance my final project in the CS 320: Software Testing, Automation, and Quality Assurance course at SNHU. I created this project throughout the entire semester in CS-320. In this course, we were tasked to create several java functions that allow for the interfacing with an arraylist to store information about contacts, tasks, and appointments.

Why This Artifact

As far as the possible artifacts I was able to choose from went, the final project from CS-320 was the artifact that had the most room for building something new. The original final program from CS-320 was a very small java program that had no user interface. I chose this for the "Algorithms and Data Structure" category, because this was the artifact that directly dealt with storing data through specific structures the most out of the valid artifacts I could choose from.

Enhancements Made

The original program had functions to allow the user to add new entries to three different lists: the contacts, tasks, and appointment lists. Each entry had to have multiple fields correctly entered. For instance, with the contacts list, the user had to correctly enter a valid first name, last name, phone number, and address. If anything was detected to be invalid, the functions wouldn't let the user create the new entry. The original project never had us create any means to interact with the functions in the program, and the original goal was just to test using JUnit tests that the functions worked correctly.

Because of the lack of any means for a user to interract with the functions, I instantly knew how I could modify the artifact to enhance the program. I created a fully functional user interface to interact with the functions, to allow the user to create and modify the entries in the lists. I created an executable Java program using JavaFX for the user interface. The program allows the user to add, remove, and modify entries in the three lists, and save those modifications to a file, and load a file of an existing list to modify.
