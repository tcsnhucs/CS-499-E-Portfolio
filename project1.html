Enhancement 1 - Software Design and Engineering

About This Artifact

For my first enhancement, I chose to enhance my final project in the CS-360: Mobile Architecture and Programming course at SNHU. I created this project throughout the entire semester in CS-360, during my second to last semester at SNHU. In this course, we were tasked to create a fully functional android application, and were asked to fulfill several different requirements related to one of three different themes. The app theme I chose was to create an application that would allow a user to create and view scheduled events in order to improve productivity for the user. For both the original version of this artifact, and the final version I used Android Studio to create my app, with the code being written in Java.

Why This Artifact

For all of these enhancements, we were tasked to choose the most fitting artifact from a previous course at SNHU to showcase our proficiency in a certain topic. This artifact was chosen for the topic of Software Design and Engineering because I felt this was the artifact that stood out the most on its own as a complete product. Most of the artifacts I could choose from were parts of a whole, and couldn't function as their own application without much more work done to them, but the android application I created for CS-360 was a complete android app that stood out on its own as a full product.

Enhancements Made

Since the app itself was mostly in a complete state, the majority of my enhancements were to improve the functions of this application and to give new features that were sorely missing. I feel the most important of these new features was the previous lack of any functional notification system, which is almost essential for an app like this. Another one of my most important enhancements was to add a filter/search system, make it so that overdue events automatically get deleted and generally improve the user interface, which all were accomplished.

In my original plans, I decided to leave the vague prompt "improve user interface" in my plans. I did this because I wanted to allow myself to decide how I should accomplish that further in the planning stage. In the end, I accomplished that by simplifying the screen layout, and removing the home screen where you selected between the calendar screen and events screen, replacing the menu with a header where you can swap between the two without leaving to a main screen. I also redesigned the user interface, so that there was a consistent purple background across the entire app, as the white background before clashed with the theming of the app, and I felt generally made the app look less professional.
